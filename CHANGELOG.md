# 更新日志

## v1.1.2 - 2026-07-13

### 兼容修复

- 适配 `wham/usage` 只返回历史用量、不再返回额度窗口的结构，刷新时不再报解析失败。
- 适配 `primary_window.limit_window_seconds=604800` 且 `secondary_window=null` 的结构，正确映射为每周额度。
- 服务端未提供短期或每周额度时，界面显示 `--` 和 `未提供`。

- 主界面和桌面挂件将原“5 小时”文案调整为“短期”，避免固定暗示旧额度窗口。
- 账号列表隐藏短期额度和短期重置列，只保留每周额度和每周重置列。
- 桌面额度挂件改为每周额度单指标显示，并下调主窗口默认打开尺寸。

### 已验证

- 主程序 Debug 构建通过，0 警告 0 错误。
- 测试项目独立输出目录构建通过，测试 DLL 输出“全部测试通过”。
- Release 发布配置 `FolderProfile` 已完成验证，产物为 win-x64 自包含单文件。

## v1.1.1 - 2026-07-12

### 兼容修复

- 适配官方 Codex 升级为 ChatGPT Codex 后的进程名称变化，停止和启动操作可正确处理新版 `ChatGPT`/`Codex` 窗口。
- 关闭 Codex App 时避免误伤 `codex-command-runner-*` 等命令执行辅助进程。
- Roaming 附加目录调整为可选安全检查，缺失时不再显示为异常。
- 增加 ChatGPT/Codex Roaming 候选目录识别，兼容新版桌面应用目录变化。

- 用量刷新改为优先使用 `https://chatgpt.com/backend-api/wham/usage` 返回的结构化数据。
- 使用 WebView2 登录态从 `/api/auth/session` 临时取得前端会话授权，并携带 `Authorization: Bearer ...` 请求用量接口。
- 账号列表新增单账号 `用量` 刷新按钮。
- 账号列表新增 `Web` 按钮，可重新打开指定账号的 ChatGPT WebView2 登录窗口。
- 用量刷新失败日志增加 HTTP 状态、session 状态和响应预览，便于定位 401 或接口返回格式变化。

### 界面与打包

- 修复打包后 exe、任务管理器和窗口图标未正确显示 Logo 的问题。
- 扩宽账号列表短期重置、每周重置和操作列，避免日期时间显示被截断。
- 重置时间增加悬停提示，便于查看完整时间。

### 已验证

- 新版 ChatGPT Codex 启动和停止已完成手动验证。
- Logo 在任务管理器和发布目录 exe 中已完成手动验证。
- `wham/usage` 接口授权刷新已通过两个账号手动验证。
- 主程序 Debug 构建和 Release 发布已完成验证。

## v1.0.0 - 2026-07-07

### 核心能力

- 支持添加 ChatGPT Web 账号，并保存独立 WebView2 登录目录。
- 支持接管当前 Codex App 登录态，生成账号 auth 快照。
- 支持更新当前账号快照。
- 支持在已接管账号之间切换。
- 切换流程固定为：关闭 Codex App，写入 live `auth.json`，自动启动 Codex App。
- 支持启动和停止 Codex App。
- 支持刷新账号用量并展示额度信息。

### 账号管理

- 支持重复账号覆盖确认。
- 支持账号重命名。
- 支持打开 App 快照目录。
- 支持打开 Web 登录目录。
- 支持删除非当前本地账号。
- 删除入口使用危险色 UI，并禁止删除当前使用、切换中或接管中的账号。

### 界面调整

- 账号列表优化为默认窗口尺寸下优先显示核心列。
- 左侧区域从无行为导航改为状态摘要。
- 右侧区域保留 Codex App 状态、当前账号、最近操作和安全检查。
- 操作日志支持清空。
- 底部支持打开本地数据目录。

### 设计收敛

- 固定使用 `%USERPROFILE%\.codex\auth.json` 作为唯一切换目标。
- 移除 Auth-only、App-only、App+Runtime 等模式选择。
- 移除 Token 模块。
- 移除 rollback 流程。
- 移除 Roaming/MSIX 替换逻辑。
- 移除宿主保护模式和模拟切换入口。
- 不切换 `.codex\sessions`。

### 已验证

- 添加账号、重复覆盖、接管、更新、真实切换通过手动验证。
- 切换后自动重启 Codex App 通过手动验证。
- 账号列表新布局、账号管理菜单、删除项危险色 UI 通过手动验证。
- 左侧状态摘要通过手动验证。
