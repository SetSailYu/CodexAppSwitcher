# 更新日志

## v1.1.0 - 2026-07-09

### 新增能力

- 新增桌面额度悬浮球，显示当前账号 5 小时额度和每周额度。
- 悬浮球支持手动刷新当前账号用量。
- 新增托盘后台状态图标，主窗口关闭后程序可继续在后台运行。
- 托盘菜单支持打开主窗口、显示或隐藏额度挂件、退出程序。

### 界面优化

- 主窗口改为两栏布局：主内容区和右侧状态详情区。
- 账号列表操作列固定显示，避免默认窗口宽度下操作按钮被遮挡。
- 当前使用账号改为整行背景高亮。
- 桌面额度悬浮球缩小并加入外圈进度显示。
- 托盘右键菜单改为暗色主题，菜单项带图标，退出项使用危险色。
- 托盘图标改为软件 Logo。
- 托盘菜单顶部状态文字已移除，只保留可操作项。

### 已验证

- 账号列表固定操作列和当前账号整行高亮已完成构建验证。
- 桌面额度悬浮球、手动刷新和关闭主窗口后保留挂件已完成构建验证。
- 托盘后台入口、Logo 图标和右键菜单 UI 已完成构建验证。

## v1.0.0 - 2026-07-07

### 核心能力

- 支持添加 ChatGPT Web 账号，并保存独立 WebView2 登录目录。
- 支持接管当前 Codex App 登录态，生成账号 auth 快照。
- 支持更新当前账号快照。
- 支持在已接管账号之间切换。
- 切换流程固定为：关闭 Codex App，写入 live `auth.json`，自动启动 Codex App。
- 支持启动和停止 Codex App。
- 支持刷新账号用量并展示额度信息。

### 账号管理

- 支持重复账号覆盖确认。
- 支持账号重命名。
- 支持打开 App 快照目录。
- 支持打开 Web 登录目录。
- 支持删除非当前本地账号。
- 删除入口使用危险色 UI，并禁止删除当前使用、切换中或接管中的账号。

### 界面调整

- 账号列表优化为默认窗口尺寸下优先显示核心列。
- 左侧区域从无行为导航改为状态摘要。
- 右侧区域保留 Codex App 状态、当前账号、最近操作和安全检查。
- 操作日志支持清空。
- 底部支持打开本地数据目录。

### 设计收敛

- 固定使用 `%USERPROFILE%\.codex\auth.json` 作为唯一切换目标。
- 移除 Auth-only、App-only、App+Runtime 等模式选择。
- 移除 Token 模块。
- 移除 rollback 流程。
- 移除 Roaming/MSIX 替换逻辑。
- 移除宿主保护模式和模拟切换入口。
- 不切换 `.codex\sessions`。

### 已验证

- 添加账号、重复覆盖、接管、更新、真实切换通过手动验证。
- 切换后自动重启 Codex App 通过手动验证。
- 账号列表新布局、账号管理菜单、删除项危险色 UI 通过手动验证。
- 左侧状态摘要通过手动验证。
